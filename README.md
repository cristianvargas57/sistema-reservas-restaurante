# sistema-reservas-restaurante
Proyecto final desarrollado en Java utilizando Netbeans.

# Descripción
  Este sitema permite gestionar las reservas de un restaurante, controlando la disponibilidad de mesas, el número de comensales y el cálculo de los costos de los menús seleccionados.

# Funcionalidades 
- Registrar reservas de clientes.
- Validar un maximo de 4 personas por mesa.
- Consultar disponibilidad de mesas.
- Seleccionar entre diferentes tipos de menú.
- Aplicar precios de acuerdo a las horas del día.
- Calcular el costo total de la reserva.
- Mostrar las ganancias acumuladas del día.
- Condultar que clientes consumieron cada tipo de menú.
